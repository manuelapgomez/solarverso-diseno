# Skill: Figma to React Engine
Propósito: Traducir nodos de Figma a código limpio y funcional en React/React Native.
- Requisito: Acceso al servidor MCP de Figma configurado por el usuario.
- Herramienta clave: Usa `get_code_connect_suggestions` para escanear el diseño y verificar si el componente ya existe en nuestra carpeta `/src/components` antes de inventar uno nuevo.
- El output debe ser siempre modular, responsivo y sin lógica de backend compleja (mockea los datos si es necesario).
