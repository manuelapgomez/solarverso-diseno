# Regla: UI de Puerto y Camiones
- **Vista Puerto:** Renderizar un grid de "Containers". Las tarjetas son rectangulares planas, pero mantienen la lógica interna de mostrar los "Slots" adentro.
- **Vista Camiones:** Renderizar tarjetas que representan camiones. Deben contener internamente sus "Slots", e incluir visualmente: Fecha de salida, Fecha ETA a la minigranja, y un identificador/guía de Ruta.
