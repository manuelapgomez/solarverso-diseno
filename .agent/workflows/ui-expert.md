# Workflow: /ui-expert
**Descripción:** Activa el análisis avanzado de diseño, investigación profunda y refactorización de código.

**Pasos de ejecución cuando el usuario usa `/ui-expert [prompt/imagen]`:**
1. **Análisis del Problema:** Revisa la imagen o el requerimiento. Identifica cuál es el verdadero problema de fondo (ej. desorden visual, flujo confuso, falta de contraste).
2. **Deep Research (Investigación):** Define qué teorías de diseño, leyes de UX o referentes visuales son los más adecuados para este caso específico y por qué.
3. **Design Walkthrough (Propuesta):** Presenta al usuario tu diagnóstico. Explica: "Basado en [Teoría/Referente elegido], el problema es X. Mi propuesta para solucionarlo es Y, aplicando Z".
4. **Implementation Plan:** Lista los archivos de código que modificarás (Tailwind, CSS, Componentes) para lograr esa visión, respetando la consistencia del proyecto.
5. **Pausa de Aprobación:** Pregunta: "¿Estás de acuerdo con este enfoque de diseño y el plan de implementación?".
6. **Ejecución:** Una vez aprobado, programa los cambios.
