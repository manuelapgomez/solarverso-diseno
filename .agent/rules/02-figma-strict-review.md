# Regla: Revisión Estricta para Figma
1. NUNCA generes, modifiques ni elimines código React de forma automática tras leer un diseño de Figma.
2. Siempre debes generar un `Implementation Plan` detallando qué componentes crearás y cuáles reutilizarás.
3. Debes esperar la aprobación explícita del usuario antes de tocar el código.
